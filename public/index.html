Search for beer name

GET: /beer?name=beername

Returns: JSON object

[
    {
        "id": id,
        "name": beer name,
        "description": beer description,
        "first_brewed": when beer was first brewed
    }
]

Rate a beer

POST: /ratebeer?id=beerid

Include JSON object in the body

{
    rating:range 1-5,
    comment:comment
}

Returns: JSON object

[
    {
        "id": id,
        "success": indicates success,
        "error": if success fails it will display an error
    }
]
